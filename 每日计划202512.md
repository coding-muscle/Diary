**1. 当天板块涨幅在市场前 3 位的题材**

**2. 板块内涨停（或大涨 >5%）数量 ≥ 2–3 只**

**3. 板块成交量放大（板块量能同比前一周上升）**

一涨停二放量三回调四缩量



① 涨幅排序：在当天、近 3 天、近 5 天排前列



② 换手率要温和（6–25% 最好）

- 换手太低＝没有资金推动
- 换手太高＝容易见顶（高度票才适合高换手）



③ 形态强度：股价在均线上方（尤其是5日/10日/20日）

一个强势股至少满足：

- **股价 > 5日均线**
- **5日、10日均线最好拐头向上**
- **MACD 不死叉**

📌 **“三线开花”=5日、10日、20日均线呈多头趋势**



④ 量价关系健康（量增价涨最强）

强势股必须具备：

- 放量上涨（最健康）
- 缩量回调（良性）
- 不大幅破位（支撑有效）

典型强势量价组合：

> 放量 → 上涨 → 缩量回踩 → 再放量突破



⑤ 市值结构优（中小市值最容易涨得快）

在同一板块中：

- 流通市值 30–100 亿：**最容易成为强势股**
- 100–300 亿：上涨但速度慢



🍀 **第三层：资金流向筛选（很关键）**

**① 主力净流入 > 0（最好连续 2–3 天）**

资金是最重要的。

只要出现：

- 主力流入
- 超大单流入
- 多次回流

**② 龙虎榜：机构席位/强势游资是否上榜**

不一定要龙虎榜，但如果有，则强度更高。

例如：

- 章盟主
- 迪士尼（福建涨停敢死队）
- 作手新一
- 上海溧阳
- 葛卫东相关席位（机构）

🍀 **第四层：过滤风险项（避免踩雷）**

强势股必须排除这些类型：

- 最近连续亏损
- 高位放天量
- 出现利空公告
- 大股东减持
- 被监管关注
- 连续缩量阴跌



一个股票满足 7 条以上，就可以重点关注：

1. 所在板块当日强度（涨幅前 1–3）
2. 板块有多只强势股（≥2–3 只涨停/大涨）
3. 个股涨幅排名靠前（1 日 / 3 日 / 5 日）
4. 换手率 6–25%
5. 股价站上 5/10/20 日均线
6. 多头排列或接近形成
7. 放量上涨、缩量回调
8. 流通市值 30–100 亿最优
9. 主力资金连续流入
10. 无重大利空（减持、亏损、监管）

满足越多，强度越高。

- 当然可以，我会用**最清晰、可实战应用**的方式讲“量价关系”。你可以把它理解为：**价格 = 情绪，成交量 = 力量。**
   量价关系其实就是在分析——市场的力量是否支持现在的方向。

  ------

  # 🎯 **一、量价关系的四大核心模型**

  下面四个是最基础、最关键、最常用的量价关系：

  ------

  ## **① 价涨量增 → 健康上涨（强趋势）**

  含义：
   上涨 → 有更多资金愿意追
   量能增加 → 资金真的进来了

  这是一段上涨行情中最健康的结构：**趋势确认 + 资金确认**。

  典型表现：

  - 阳线放量
  - 突破压力位伴随量能放出
  - 主升浪常常就是“价涨量增”

  ------

  ## **② 价涨量缩 → 弱反弹 or 上涨衰竭**

  含义：
   价格上涨，但没有新资金参与。
   说明上涨缺乏持续性。

  常见两种情况：

  - **下跌后的反抽**（弱反弹，没有主力真回流）
  - **上涨末端的衰竭**（趋势疲软，随时可能被抛压打断）

  关键判断：
   看 K 线是否接近压力位；如果是 → 很容易被压回去。

  ------

  ## **③ 价跌量增 → 下跌加速（主力撤退、恐慌盘）**

  含义：
   股价下跌时量能放大，说明：

  - 有资金主动卖出
  - 跌得越多卖得越快
  - 恐慌盘涌出

  通常是 **趋势下跌的确认信号**
   特别是在跌破均线、跌破箱体、跌破支撑位时。

  ------

  ## **④ 价跌量缩 → 缩量盘整 or 下跌减弱**

  含义：
   股价跌，但量越来越少 → 说明：

  - 砸盘动力不足
  - 卖压疲软

  常见于：

  - 下跌末期的企稳
  - 下跌后的横盘（筑底信号之一）

  经典应用：
   **缩量阴跌 + 关键支撑位 → 底部可能形成**

  ------

  # 🎯 **二、量价八种进阶组合（更实战）**

  除了四大模型，实战中常见八种量价组合：

  ------

  ## **1. 放量突破（最强买点）**

  价格突破前期压力 + 明显放量
   → **主力在发动攻击、趋势真正开始**

  特点：

  - 量能必须超过近 5–10 日平均量
  - 越关键的压力位，越要看量

  ------

  ## **2. 缩量回踩（极强标志）**

  突破后的小幅回踩，量能显著缩小
   → 多头力量强，空头不敢砸

  典型结构：
   突破（放量） → 回踩（缩量） → 再拉升（放量）

  ------

  ## **3. 放量滞涨（顶部警告）**

  量能放得巨大，但价格不再上行
   → 资金开始出货，顶部有风险

  形态表现：

  - 实体小，但量很大
  - 长上影 + 天量（很危险）

  ------

  ## **4. 半分位放量突破失败（假突破）**

  突破时放量，结果涨不动还跌回去
   → 多头力量不够 or 主力借突破出货

  后果：

  - 很容易形成假突破陷阱
  - 第二天往往继续下跌

  ------

  ## **5. 缩量横盘（主力控盘、筹码被锁定）**

  量小、波动小、三角形收敛
   → 主力在等契机→ 一旦放量突破 = 强势启动

  ------

  ## **6. 高位缩量（多头疲软）**

  在高位缩量横盘 → 表面强，但难持续
   → 若跌破关键均线，则下跌风险巨大

  ------

  ## **7. 下跌放量长下影（恐慌砸盘后的抢筹）**

  下影越长，越说明低位有人抢
   → 若次日高开 + 放量，可以确认反转

  ------

  ## **8. 涨停板量能判断（连板关键）**

  涨停当日两种情况很关键：

  ### （1）换手板（量大）

  → 连板概率高
   → 主力洗盘、锁筹

  ### （2）秒板（量小）

  → 强一致性，但二板容易分歧

  ------

  # 🎯 **三、量价关系怎么结合到实战？**

  下面给你具体的“量价判断 checklist”，你可以在看任何股票时直接用：

  ------

  ## ✔ **实战决策：上涨是否可靠？（看 3 点）**

  1. 是否放量？
  2. 是否突破压力？
  3. 放量是否持续？

  三者同时满足 → 主升段起飞概率很大。

  ------

  ## ✔ **实战决策：下跌是否危险？（看 3 点）**

  1. 是否放量下跌？
  2. 是否跌破关键均线？
  3. 是否跌破重要支撑？

  满足 1+2 → 大概率进入下跌趋势。

  ------

  ## ✔ **实战决策：买点确认方式**

  最稳的是：

  **放量突破后 → 缩量回踩 → 再放量起涨**

  这是一套趋势交易中最安全的结构。

  ------

  # 🎯 **四、量价关系决定的三种核心买点**

  你可以直接套用：

  ### 👉 买点 1：放量突破压力

  （趋势开始）

  ### 👉 买点 2：缩量回踩中轨、均线、箱体

  （洗盘完成）

  ### 👉 买点 3：突破回踩的再次放量

  （趋势确认）

  ------

  # 🎯 **五、中国 A 股特别适用的量价规律**

  A 股是典型的“情绪市 + 板块联动强”的市场，所以：

  - **放量涨停（换手板） → 次日溢价高**
  - **缩量涨停（一致板） → 次日溢价不稳定**
  - **放量跌停 → 资金出逃明显，下跌未完**
  - **缩量跌停 → 缺乏抄底力量，次日多为继续弱势**

  量价是连板交易与情绪周期最核心的基础。

  ------

  # **① 试探性拉升（探路拉升 / 测压拉升）**

  ### **特征（你写的 + 专业补充）**

  1. 小资金推升 2%–4%，速度不快。
  2. 上方抛压（卖单）略放出，股价容易打回到 +1% 附近。
  3. 若抛压较大甚至回到开盘价附近。
  4. 当天多次重复这一套“推升→回落→再推升”动作。

  ### **主力目的**

  - **探路情绪**：测试上方套牢盘、浮筹是否多。
  - **观察跟风盘**：看有没有散户愿意跟。
  - **为建仓找价格**：反复拉升制造价格锚定，方便低吸。

  ### **识别方法**

  - 放量不大，大多**1~5% 的温和放量**。
  - 多次出现 **上影线不长** 的小幅冲高。
  - 分时波动小而密集。

  ### **风险 / 应对**

  - 适合观察，不适合追涨。
  - 若随后出现 **试探 → 转强 → 平台突破**，才是真信号。

  ------

  # **② 攻击性拉升（强势拉升 / 主升拉升）**

  ### **特征**

  - 一般 **突破关键点位**（箱体上沿 / 前高 / 均线密集区）。
  - 明显放量，涨幅通常 **单日 5%–7% 以上**。
  - 分时几乎“一路向上”。

  ### **主力目的**

  - **快速脱离成本区**，减少抛压干扰。
  - **点火带动情绪**，吸引跟风盘。
  - 在确定性强时进入主升浪阶段。

  ### **识别方法**

  - K 线中阳 / 大阳线。
  - 成交量明显放大为 **近期数日最大量**。
  - 龙头股通常都用这种“攻击性拉升”。

  ### **风险 / 应对**

  - 攻击性拉升才是比较安全的介入点。
  - 看资金：若成交量急剧萎缩是诱多；若持续放量是趋势成立。

  ------

  # **③ 对倒式拉升（虚假拉升 / 自买自卖拉升）**

  ### **特征**

  - 在某个周期内（例如 30 分钟或全天）反复上下拉。
  - K 线出现 **多根长上影线**。
  - 成交量看似放大，但盘口单子很规整（对敲味道）。

  ### **主力目的**

  - **制造活跃度**，吸引散户以为有人在进场。
  - **拉高出货**或给大资金提供更好的卖点。
  - **做换手外观**，让图形变好看。

  ### **识别方法**

  - 多个长影线，但实体不大。
  - 分时波动急促，且拉升突然、回落也突然。
  - 资金大多 “冲上去又冲下来”。

  ### **风险 / 应对**

  - **不要追！高风险信号。**
  - 多数对倒出现在：
    - 利好前夕的“演戏”
    - 主力减仓期
    - 假突破

  ------

  # **④ 脉冲式拉升（消息刺激拉升）**

  ### **特征**

  - 柱状脉冲上攻，一根或几根大阳。
  - 常见于 **政策 / 行业利好 / 公司公告** 当天。
  - 分时呈现 **一笔一笔大单往上打**。

  ### **主力目的**

  - 利好推动价格快速靠近“合理价值”。
  - 并不一定是主力主动攻击（可能是市场情绪推动）。

  ### **识别方法**

  - 当天全行业大涨。
  - 该票跟随异动 → 换手明显扩大 → 情绪一致。

  ### **风险 / 应对**

  - 若利好是真实长期（新能源 / 算力 / 机器人）→ 可追强。
  - 若是一次性短期利好 → 多是一天行情 → 次日注意高开低走风险。

  ------

  # **⑤ 诱多式拉升（假拉升 / 高位出货拉升）**

  ### **特征**

  - **高位放量 → 冲高 → 快速回落**。
  - 上影线长。
  - 分时中后段突然拉升吸引散户追买。

  ### **主力目的**

  - **制造突破假象**，让散户接盘。
  - 完成出货、控盘降低。
  - 通常出现在 **趋势末期** 或 **双顶结构**。

  ### **识别方法**

  - 大阳线后伴随长上影线，量价背离明显。
  - 量放太大：天量见天价。
  - 次日往往无法延续。

  ### **风险 / 应对**

  - **不要追高！**
  - 若出现长上影 + 天量，尤其在前高附近 → 大概率筑顶。

  ------

### 竞价复合模式

- 下面我帮你把 **“竞价复合模式”** 系统、清晰地讲一遍（这是短线交易里非常实用、胜率较高的竞价筛票与预判方法）。

  我会从 **定义 → 作用 → 核心逻辑 → 实战模式类型 → 具体操作步骤** 五个部分讲清楚。

  # ✅ 一、什么是“竞价复合模式”？

  一句话：
   **竞价复合模式 = 把竞价的多个关键信号叠加确认，从而判断当天是否有“强势票”值得交易的模式。**

  复合的意思是 **不是只看一个因素，而是多个竞价信号共同出现 → 形成强确认**。

  常见复合维度包括：

  - 竞价强度（上涨幅度、竞价分时形态）
  - 竞价量能（放量与否）
  - 竞价资金方向（主动买入、量比、外盘、资金流入）
  - 当日板块情绪（是否有强题材）
  - 前一日的形态（首板、二板、断板、回调、筹码结构）
  - 龙头确认（龙头竞价强，带动同题材）

  **最终目的是：在 9:25 之前找出当天最强的一批票。**

  ------

  # ✅ 二、为什么竞价复合模式很重要？

  因为：

  - 9:25 的竞价反映了 **第一批资金的态度**
  - 是当日 **最真实的强弱分水岭**
  - 强势票在 9:25 **一定会有信号**（高开、抢筹、放量）
  - 弱势票在 9:25 **直接露馅**（低开、缩量、无人接）

  简单说：
   **竞价强 → 资金敢买 → 当天有走强可能
   竞价弱 → 资金不敢买 → 当天大概率走弱**

  竞价复合模式，就是把这些信号组合在一起，提高胜率。

  ------

  # ✅ 三、竞价复合模式的核心逻辑（非常关键）

  三个字：

  ### **抢、强、顺**

  ### 1）抢：资金抢筹

  - 竞价放量
  - 主动买单占比高
  - 外盘大
    说明有资金急着要买。

  ### 2）强：竞价强度

  - 高开 >3%
  - 若是强势情绪票，高开>5%更强
  - 分时红柱明显
  - 分时直接拉升

  这些都说明“强”。

  ### 3）顺：顺着当天的主线

  - 强题材竞价集体高开
  - 龙头竞价很强，支线同步跟随
  - 市场情绪转强（上一日涨停多、连板率高）

  **强 + 抢 + 顺**
   = 当日大概率强势股。

  ------

  # ✅ 四、竞价复合模式 —— 六大常见复合形态

  （每一种都有强操作意义）

  ------

  ## ⭐ 模式 1：高开 + 放量 + 主线题材（最强模式）

  条件：

  - 高开 3–6%
  - 竞价量 > 前一日同时间量 2 倍以上
  - 属于市场主线题材

  适用情景：

  - 龙头或前排强势股
  - 有势能、有情绪、有资金

  成功率：最高
   打法：竞价直接上、或集合竞价尾部挂单。

  ------

  ## ⭐ 模式 2：高开 + 竞价阶梯式推升（资金持续入场）

  表现：

  - 分时 9:15–9:25 不断被资金推升
  - 竞价从 +1% 推到 +4% / +5%
  - 明显有资金在“计划性建仓”

  代表：
   **主力在竞价试探 → 越买越多 → 越高越买**

  胜率高，尤其是强热点。

  ------

  ## ⭐ 模式 3：一字强竞价（情绪高潮期）

  表现：

  - 一字板挂单巨大（封单占比大）
  - 题材强势情绪

  适用：

  - 市场龙头
  - 次日高溢价概率高

  风险：

  - 不能盲追封高，适合买在分歧转一致的首板或二板。

  ------

  ## ⭐ 模式 4：高开弱转强模式（高开低走后强资金承接）

  表现：

  - 高开 → 短暂回落 → 竞价尾部被资金重新拉起
  - 尾盘价格接近竞价高位
    意义：
  - **冲高后不砸，说明承接强**

  适合在开盘 1 分钟附近低吸。

  ------

  ## ⭐ 模式 5：低开高走（逆势抢筹）

  表现：

  - 本应弱，却被竞价资金推起来
  - 分时明显抢筹

  典型：

  - 利空预期过度
  - 强势票洗盘

  玩法：低吸最好。

  ------

  ## ⭐ 模式 6：板块一致性竞价（资金锁定的“强共振”）

  表现：
   例如：

  - 航空板块全部高开
  - 半导体全板块 3–5% 高开
  - 大消费全体高开

  意义：

  - 整个板块被资金点火
  - 龙头大概率走强

  ------

  # ✅ 五、竞价复合模式 —— 实战操作步骤（最重要）

  以下是短线交易者每天 9:15–9:25 的操作 SOP：

  ------

  ## **📌 ① 9:15–9:20 —— 初筛（看强弱）**

  看四个指标：

  - 开盘涨幅（强弱一眼分明）
  - 竞价量（放量最关键）
  - 分时形态（是否推升）
  - 题材是否主线（顺势）

  筛选出 *10–20 只候选票*

  ------

  ## **📌 ② 9:20–9:24 —— 精筛（看是否“复合”）**

  重点观察：

  - 是否 **高开 + 放量 + 主线** 三者复合
  - 龙头是否竞价强（龙头强才敢买支线）
  - 是否有资金抢筹（外盘大于内盘）
  - 竞价成交金额是否超过昨天同时间量

  筛到只剩 *3–5 只最强票*

  ------

  ## **📌 ③ 9:24–9:25 —— 下单（卡位置）**

  根据模式类型：

  - 若强势复合（模式 1 或 2）
    → **集合竞价尾单直接挂买**
  - 若“弱转强”模式
    → 等开盘 1 分钟承接后买
  - 若强板块一致性
    → 龙头优先买，不买杂毛

  ------

  ## **📌 ④ 9:30–10:00 —— 强弱确认**

  观察：

  - 是否放量突破
  - 是否给回踩机会
  - 是否被资金砸盘
  - 板块是否跟随

  若符合预期 → 拿
   不符合 → 立刻撤退（尤其是首阴）

  ------

  ## 📌 含义

  - 情绪亢奋
  - 很可能出现“高开低走”

  ## 🟨 建议

  **不开仓，不减仓，观察 10–20 分钟分时趋势再做决策。**

  ## 📌 含义

  - 大盘因外围因素低开（非内部风险）
  - 没有行业利空
  - 个股基本面正常

  ## 🟩 建议

  敢买的人少，但这是 **最强的逆向机会点之一**。

  ------

  # **⑥ 早盘高开 → 多为减仓结构**

  ### 📌 早盘高开常见 2 种情况：

  - 利好兑现

  ### 📌 特别危险：

  “高开 → 量小 → 快速回落”＝明显的诱多结构

  ## ❌ 建议

  早盘高开时不追，甚至需要 **减仓锁定利润**。

  ------

  # **⑦ 下午大涨 → 要减仓（兑现）**

  ## 📌 A 股特点

  下午的上涨经常是：

  - 主力拉高对倒做图形
  - 下午冲情绪
  - 第二天高开低走预期强

  ## 🟥 建议

  **下午大涨的票不要抱，兑现更安全。**

  # **⑧ 下午大跌 → 次日买（极强逻辑）**

  ### 📌 为什么？

  下午大跌一般是：

  - 市场恐慌宣泄
  - 临近收盘的被迫卖盘
  - 机构挤压尾盘（洗盘套路）
  - 情绪崩溃点

  ### 📌 次日常见走势

  - 大概率低开高走
  - 低吸很安全
  - 是高手常用“情绪转折点”模型

  ## 🟩 建议

  **隔日低吸，胜率很高（尤其情绪见底时）。**

- **“同店销售”（Same-Store Sales，简称 SSS）**
  是零售行业中一个非常关键的指标，用来衡量“老店”的真实经营状况。

  ------

  # ✅ **一句话解释**

  **同店销售 = 已开业满一定时间（通常 1 年以上）的门店，在同一时间周期的销售额同比变化。**

  ------

  # 🧠 **为什么要看“同店销售”？**

  因为零售企业会不断开新店，如果只看总营收，会被新店“拉高数据”，掩盖老店经营变差的问题。

  同店销售只关注“老店”，这样可以判断企业经营是否真正变好。

  ------

  # 📌 **一个简单例子**

  某零售企业 2024 年总收入同比增长了 20%。
   但如果：

  - **同店销售只有 +1%** → 说明大部分增长来自“新店开张”，老店增长乏力
  - **同店销售为 -5%** → 老店在下滑，可能行业不景气或公司竞争力下降
  - **同店销售为 +10%** → 老店经营非常健康，企业质量较强

  ------

  # ✨ **同店销售的意义**

  1. **判断企业的内生增长**（不是靠开店硬撑）
  2. **检验销售效率变化**
  3. **评估消费景气度**
  4. **是零售 & 餐饮行业最重要的运营指标之一**

  ------

  # 📍 **哪些行业特别看重同店销售？**

  - 商超行业（永辉、沃尔玛）
  - 便利店（美宜佳、711）
  - 餐饮（海底捞、麦当劳、星巴克）
  - 服装（安踏、优衣库）
  - 美妆集合店
  - 家居卖场（宜家）

  ------

  # 📈 **投资时怎么看？**

  同店销售通常被认为是：
   **> +5%：较强
   0%～+3%：正常
   负数：需警惕行业或品牌竞争力下滑**
