2026.3.7 煤化工

纵观一周的盘面，还是能够发现整个市场的情绪依然被石油所裹挟，周五国际石油期货的大幅上涨还是会让人联想到煤化工的炒作。

我国是"富煤贫油少气"的资源禀赋，石油短缺促使发展新能源产业和煤化工替代战略。尽管在新能源的格局下，石油缺口依然是比较大的，现在油和煤的价差，极有可能会导致国家对于煤炭的重视，如果对于原油板块把握不了的朋友，不妨从煤炭入手。

关注几个方向：

煤制油：中国神华、兖矿能源、潞安环能 

煤制气：大唐发电、中煤能源、广汇能源、新奥股份、特变电功

煤制烯烃/乙二醇：宝丰能源、华鲁恒升、金牛化工



说起煤炭，作为产煤大省的新疆定位其实非常重要。去年的十五五规划中，新疆在煤化工投资规划中占比达40%，5年投资金额预计达6000亿~1万亿。

关注以下几个方向

煤设备、EPC（新疆项目建设将带动设备需求，设备投资额将占到60%）

EPC工程：东华科技（弹性较高）、中国化学

专用设备：三维化学、航天工程（由于商业航天的炒作，需注意筹码结构问题）

  在荣格看来，“子承父业”的本质，往往是**家庭无意识（Family Unconscious）**对个体灵魂的“占据”。他会从以下几个维度来拆解：

  ### 1. 家庭“情结”的传递：未完成的人生任务
  荣格认为，家族中未解决的冲突、未被满足的野心、或者被压抑的阴影，会形成强大的**情结**。这些情结像隐形的磁铁，会吸附后代的生命力。
  *   **解释：** 如果父亲在职业生涯中感到“未竟”或“匮乏”，这种能量会通过无意识传递给儿子。儿子继承家业，表面上是自主选择，实际上可能是被父亲的**“未完成情结”**所驱使。荣格会说，这不是儿子在过自己的人生，而是**“父亲的灵魂”**在借着儿子的生命继续赶路。

  ### 2. 父亲原型与人格面具的强制继承
  在荣格心理学中，**父亲原型**代表着权威、秩序、社会规则以及在外部世界生存的能力。
  *   **解释：** 当一个儿子“子承父业”时，他往往继承的不仅仅是一个职位，而是父亲精心构建的**人格面具**（Persona，即人在社会舞台上扮演的角色）。如果这份职业恰好契合了父亲原型在儿子心中的投射，儿子会感到安全；但如果这份职业与儿子的内在真实自我（Self）不符，他就会陷入一种**“神圣的附身”**状态——他表现得像个成功的企业家或工匠，但内心却感到空洞、虚假，甚至产生中年危机。

  ### 3. 集体无意识与“祖先的期待”
  荣格非常重视集体无意识。他认为，对于人类来说，“继承”是一种深层的原始意象。
  *   **解释：** 在原始部落中，儿子必须继承父亲的狩猎技巧或巫术，否则部落就会灭亡。这种**“祖先期待”**深植于我们的集体无意识中。因此，当儿子选择不子承父业时，他往往会体验到一种原始的内疚感，仿佛背叛了整个宗族。荣格会认为，这种内疚感并非来自道德，而是来自与**集体无意识的断裂**。

  ### 4. 个体化（Individuation）与“弑父”
  这是荣格理论中最核心、也最残酷的部分。他认为人生的终极目标是**个体化**——即成为那个“本应成为的、独特的自己”。
  *   **解释：** 如果子承父业仅仅是一种机械的复制，那么儿子就永远只是“父亲的延伸”，而不是一个完整的人。荣格认为，为了完成个体化，儿子必须在心理层面完成某种象征性的 **“弑父”**。
      *   这种“弑父”不是物理上的伤害，而是在精神上**击败父亲的原型**。
      *   具体表现为：儿子可能在中年时毅然转行，或者在继承家业后，对产业进行了颠覆性的、父亲不理解的改革，或者通过某种创造性的方式，在继承传统的同时，找到了属于自己灵魂的独特表达。

  ### 5. 命运的重复与解答（呼应你上一轮的提问）
  结合你上一轮提到的“命运会一直出重复的问题，如果给予解答”，荣格对子承父业的看法完全契合这一逻辑。

  荣格会说：
  > **“命运让儿子重复父亲的职业，并不是为了让儿子成为第二个父亲，而是为了让儿子在这个重复的‘课题’中，给出父亲当年没能给出的‘新答案’。”**

  如果儿子只是重复，那就是命运的僵化（神经症）；如果儿子在继承中通过意识的介入，化解了家族情结，注入了新的灵魂内涵，那么他就完成了荣格所说的 **“超越功能”** ——他在同一种命运里，完成了对命运的超越。

  ### 总结
  在荣格看来，“子承父业”是一场盛大的**象征性仪式**。它既是一份来自集体无意识的**馈赠**（让儿子无需从零开始构建社会角色），也是一个巨大的**陷阱**（让儿子迷失在父亲的人格面具中）。

  **荣格式的健康状态**是：儿子能够拿起这份“父业”，像穿上一件衣服一样，既能利用它在这个世界上行走，又清楚地知道“我不是这件衣服”，并且在必要的时候，有勇气将其脱下，去追寻那个独属于内在自性的召唤。

  对于这类人，**外倾直觉**是主导功能。这意味着：

  -   **对“可能性”上瘾**：他们对外部世界的感知方式不是看它“是什么”（感觉），而是看它“可能变成什么”（直觉）。一座老房子在他们眼里不是破旧的建筑，而是改造成网红咖啡馆的绝佳地点；一个陌生人在他们眼里不是当下的样子，而是未来能带来巨大商业价值的合作伙伴。
  -   **快速切换的注意力**：他们的精力像探照灯，不停地扫描环境中的新事物。一旦发现一个新兴趣、新概念或新人，他们会瞬间被点燃，投入极大的热情。但当这个可能性被探索完毕、失去了“新鲜的可能性”时，他们会迅速将注意力转向下一个新目标。
  -   **破坏性创造者**：荣格认为，这种类型的人往往在文明发展中扮演“开荒者”或“企业家”的角色。他们天生厌恶既定的传统和稳定的结构，因为稳定意味着没有变化的空间。他们热衷于打破旧秩序，引入全新的、有时甚至是疯狂的理念。

  ### 2. 外在表现：热情、冒险与不负责任
  外向直觉型的人通常是社交场合中的“造梦师”和“点火者”，其外在特征非常鲜明：

  -   **极其迷人的启动器**：他们拥有一种特殊的魅力，能用热情和愿景感染他人。当周围人都在犹豫时，他们能拍着桌子说“试一试又不会怎样！”他们是天生的创业者，擅长从0到1的阶段，能把一群不相干的人聚集在一起，为一个宏大的愿景兴奋不已。
  -   **道德与现实的盲点**：由于辅助功能通常是**内倾感觉**（这是他们最薄弱的功能），外向直觉型的人往往对**身体的感受、过去的教训以及世俗的道德约束**感知很弱。
      -   他们可能为了一个新项目而忽视身体健康，直到累倒才察觉。
      -   他们容易“好了伤疤忘了疼”，很难从过去的错误中吸取教训（因为过去属于感觉，他们不擅长这个）。
      -   他们可能会为了追逐新的可能性，而轻易抛弃旧有的责任、伴侣或合作伙伴，给人留下“不靠谱”“善变”或“不负责任”的印象。
  -   **厌倦固化**：一旦一个项目进入稳定期，需要重复性的维护和管理时，外向直觉型的人通常会感到“窒息”。他们会觉得这件事已经“死”了，没有潜力可挖了，于是果断抽身去寻找下一个风口。

  ### 3. 荣格在《心理类型》中的描述
  荣格对这类人的描述带有一种爱恨交织的复杂感：

  -   **正向发展**：他们是**开拓者、企业家、发明家、投机者**。在商业领域，他们是发现新大陆的冒险家；在文化领域，他们是引领潮流的先锋。很多杰出的记者、连续创业者、经纪人（善于挖掘新人）都属于这一类型。他们能让沉寂的社会流动起来。
  -   **负向发展（病态）**：如果过度发展直觉而压抑感觉，他们会变成 **“四处点火却无法收场的投机分子”** 。他们可能会陷入对“新奇感”的病态追求，不断换工作、换伴侣、换信仰，却从未真正深入体验任何一段关系或事业。荣格指出，极端情况下，这种人可能会为了追求某种“新感觉”而变得不道德，甚至做出疯狂的举动，因为他们感觉不到现实的约束。

  ### 4. 与“内向直觉型”的鲜明对比
  为了更好地理解，可以将两者放在一起对比：

  | 维度         | 外向直觉型                                             | 内向直觉型                                               |
  | :----------- | :----------------------------------------------------- | :------------------------------------------------------- |
  | **关注焦点** | **外部世界的可能性**。哪里有新机会？谁能给我新启发？   | **内心涌出的意象**。这个外部事物在我心里激起了什么原型？ |
  | **时间感**   | **横向的广度**。热衷于体验不同领域，追求多样性和变化。 | **纵向的深度**。穷尽一个意象的全部内涵，追求终极意义。   |
  | **行为模式** | 像**蝴蝶**，在万花丛中掠过，不断寻找下一朵盛开的花。   | 像**深海潜水员**，选定一个点，不断下潜，探索深渊。       |
  | **社交表现** | 社交达人，能快速与陌生人建立联系，能量外放。           | 安静、疏离，即使社交也是为了寻找能理解其内在意象的知音。 |
  | **风险**     | 浅尝辄止，无法坚持，逃避责任。                         | 脱离现实，不被理解，固执于内在幻象。                     |

  ### 5. 现代生活中的例子
  在MBTI体系中，外向直觉型对应 **ENFP**（竞选者）和 **ENTP**（辩论家）的主导功能（Ne）。

  -   **ENTP（外倾直觉+内倾思维）**：典型的“点子大王”。在头脑风暴会上，当大家还在讨论第一个方案时，他们已经蹦出了十个截然不同的方案。他们热衷于辩论和挑战现有的逻辑体系，享受在思维层面推翻重来的快感。
  -   **ENFP（外倾直觉+内倾情感）**：充满热情的人本主义者。他们能看到每个人身上的潜能，并热衷于帮助别人发掘这种潜能。他们可能频繁地尝试不同的兴趣爱好，探索不同的人际关系，追求丰富多彩的生命体验。

  ### 总结
  **外向直觉型**是一种**将全部感知力投向外部世界中的新联系、新机会与新可能性**的心理类型。

  他们是时代的**点火者**，擅长在荒原上点燃第一把火，将人们从沉闷的现状中唤醒。他们的优势在于**洞察趋势、联结资源、开启新局**；他们的弱点在于**难以深耕、回避常规、容易在现实琐碎（金钱、健康、守时）上栽跟头**。

  对于外向直觉型的人，荣格给出的成长建议通常是：**发展“感觉”功能**。学会享受当下的宁静，尊重自己的身体感受，认真对待过去的经验教训，培养对重复性事务的耐心。否则，他们可能会一生都在追逐“远方的彩虹”，却永远无法真正收获眼前的果实。

### 军师解盘

周五锂矿相关股票的大涨，让市场重新审视新能源逻辑。

一，石油板块

美伊战争是石油价格上行的最直接的原因，战争造成能源体系混乱，但是从政治角度来看，油价上涨其实符合上三常（中美俄）的利益，美国在页岩油革命后是世界第一大产油国，同时也是能源出口国；俄罗斯是典型能源型国家，30%–50%的财政收入来自能源；中国是能源投资国，能源价格上涨间接导致新能源竞争力增强。与此同时，欧盟和日本等牌桌上其他国家利益就受到损害。

二，煤化工 氢能源

3月初战争初期就持续炒作，持续观察金牛化工，宝丰能源。

氢能源两会后被兑现，但作为能源替代的方向之一，还需要持续关注。

三，黄金

目前黄金主要是两大属性的对抗，避险性VS流动性，黄金的大幅兑现是目前黄金板块调整的主要原因。

但是需要注意，近期金油上涨 ，黄金板块可能面临变盘。

重点观察紫金矿业，它目前处于平台位置，作为有色板块的大票需要观察态度。

四、新能源

宁德时代财报刺激股价创新高，加强机构布局新能源的意愿。 

比亚迪年报发布，虽然四季度报同比下降，但是环比上升，加上近期刀片电池技术突破，油电价差扩大刺激需求以及海外渗透率提升（4-5月排产加速），作为新能源板块的龙头，仍需要持续观察。

五、锂矿

三大因素造成锂矿价格上涨

1.供给端的扰动 津巴布韦减产+宁德时代项目延迟影响10万吨产能

2.四五月份排产加速 

3.反内卷政策推动行业集中度提升，

目前全球供给150万吨，增量40万吨，但是增量会产生变数，储能领域新增20万吨需求（对应300GWh产能）

重点关注以下锂矿企业：

赣锋：股价表现强过天齐，自有锂矿+马里6万吨产能，28年14-15万吨产能

天齐：全球成本最低，12万吨产能

中矿：纳米比亚 6亿利润，关联云南锗业 小金属逻辑

大中：资产重组后，股价长期横盘 湖南锂矿8万吨 四川锂5万吨，对标赣锋，重点持续观察

国城：钼矿8亿利润，锂矿扩产

天华：创业板 宁王入股 走势像赣锋 宁王入股 刚果30%股权，产能15万吨 尼日利亚远期规划10w吨

永兴：成本优势，云母矿5万吨，特钢

华友：10万吨碳酸锂 钴镍

盛新：2.5万吨碳酸锂 1.5万吨氢氧化锂 远期有9万吨规划

融捷：与比亚迪联系紧密 21年跟江特电机一起炒作过 受游资关注度特别高 去年年报金矿产能超预期

盐湖：钾 远期锂矿产能

六、总结

虽然股市最近很糟心，但是作为投资者不能浪费每一次危机。

普通投资者在市场波动的时候更要注重逻辑，不要受外围战争和美股影响，在下跌过程中观察承接，持续观察资金态度。
